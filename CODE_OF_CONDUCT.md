# Code of Conduct

## Our Commitment

We are committed to making participation in this project a respectful, harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent and collaborate in good faith.
- Give and receive feedback professionally.
- Keep discussions focused on improving the project.

## Unacceptable Behavior

- Harassment, discrimination, or hateful language.
- Personal attacks, trolling, or intimidation.
- Publishing private information without consent.
- Any conduct that is inappropriate in a professional setting.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct and may remove, edit, or reject comments, commits, code, issues, and other contributions that are not aligned with this policy.

## Reporting

If you experience or witness unacceptable behavior, open a private security/administration report in this repository. Maintainers will review and respond as quickly as possible.

## Scope

This code of conduct applies within all project spaces, including issues, pull requests, discussions, and other communication channels related to this repository.
